开发环境——debian

在 debian 中安装 clojure 

sudo apt -t bookworm-backports install clojure leiningen
  • lein 是项目管理、包管理、命令行工具

测试是否安装好clojure： 执行 clojure 

$ clojure
Clojure 1.10.2
user=>

在 user=> 后即可输入代码
