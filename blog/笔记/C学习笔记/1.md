# 例子1

例子1: 编写程序输出"hello world"

C语言是编译型语言, 编写一个C程序的步骤:

1. 使用编辑器(vscode等)编写程序编写程序源代码
2. 使用编译器将源代码编译为计算机识别的机械码(010100之类)

首先在文本编辑器中新建 `hello.c`

## 代码

> 输出 "Hello World"

```c
#include <stdio.h>

main () {
    printf ("Hello world!\n");
}
```

1. `#include <stdio.h>` 引入c语言的标准IO库(input output), 库提供了各种各样我们需要的功能, `printf` 就被包含在 `stdio.h` 库中
2. `main()` c语言从 main 部分开始执行
3. `printf ("Hello world!\n");` printf 表示输出引号内的字符串 "Hello world!"

    `\n` 是格式转换符,表示在输出 Hello world 后跳转到下一行.

## 编译程序

在命令行输入 `gcc –o myprog hello.c`

* `-o myprog` 表示编译后的程序名为 myprog

`./myprog` 运行程序
