# 队列

先进先出

## 顺序队列

### 结构体

```c
#include <stdbool.h>
#define MAXSIZE 100

typedef struct SqQueue {
    int data[MAXSIZE];
    int front, rear;
} SqQueue;
```

### 初始化

```c
SqQueue CreateSqQueue(void) {
    SqQueue q;
    q.front = q.rear = 0;
    return q;
}
```

### 判断队空

```c
bool SqQueueIsEmpty(SqQueue q) {
    if (q.front == q.rear) {
        return true;
    }
    else {
        return false;
    }
}
```

### ~~入队~~

* **先加元素再移动指针**

```c
bool EnQueue(SqQueue* queue, Element value) {
    if (queue->rear == MAXSIZE) { return false; } /*队列已满*/
    queue->data[queue->rear] = value;
    queue->rear = queue->rear + 1;
    return true;
}
```

### ~~出队~~

```c
bool DeQueue(SqQueue* q, Element* e) {
    if (q->front == q->rear) { return false; } /*空队列*/
    *e = q->data[q->front];
    q->front = q->front + 1;
    return true;
}
```

# 循环队列

一般队列多次删除以后空间不足，循环队列是为了克服假溢出这个缺点。循环队列的结构体、初始化、判断队空操作与一般队列一致。

## 入队

```c
bool EnQueue(SqQueue *q, Element data) {
    /*判断队满*/
    if ((q->rear + 1) % MAXSIZE == q->front) { return false; }
    q->data[q->rear] = data;
    q->rear = (q->rear + 1) % MAXSIZE;
    return true;
}
```

## 出队

```c
bool DeQueue(SqQueue* q, Element* output) {
    if (q->front == q->rear) { return false; } //空队列
    *output = q->data[q->front];
    q->front = (q->front + 1) % MAXSIZE;
    return true;
}
```

## 队列的计算

1. 已知 front rear, 队列元素个数为 (rear+maxsize-front)%maxsize

# 链式队列

## 结构体

```c
//链式队列的结点
typedef struct QNode {
    int data;
    struct QNode *next;
} QNode;

//链式
typedef struct {
    QNode *front;
    QNode *rear;
} LiQueue;
```

## 初始化队列

```c
void InitQueue(LiQueue *q) {
    q = (LiQueue*)malloc(sizeof(LiQueue));
    q->front = q->rear = NULL;
}
```

## 入队

```c
void enQueue(LiQueue *q, int data) {
    QNode *node = (QNode*)malloc(sizeof(QNode));
    node->data = data;

    if (q->rear == NULL) { //空队列
        q->front = q->rear = node;
    } else {
        q->rear->next = node;
        q->rear = node;
    }
}
```

## 出队

```c
bool deQueue(LiQueue *q, int *output) {
    if (q->rear == NULL) {return false;} //空队列
    QNode *node = q->front;

    //队列中只有1个结点时需要特殊处理
    if (q->front == q->rear) {
        q->front == q->rear == NULL;
    } else {
        q->front = q->front->next;
    }
    output = node->data;
    free(node);
    return true
}
```
