# 什么是线性表

多个相同类型的数据元素组成的有限序列，所有元素组合起来即是一个线性表。

线性表具体代码有2种表示方式：顺序表和链表。

# 顺序表

顺序表通过 **数组** 来实现，一个顺序表包括

* 存储表中元素的数组 data[]
* 指示数组个数的 length
*  顺序表的索引从1开始

## 结构体

```c
include <stdbool.h> //引入布尔标准库

/*SqList: sequence list 顺序表*/
#define MAXSIZE 100 //定义顺序表的最大长度

typedef struct {
    int length; //顺序表的长度
    int data[MAXSIZE];
} SqList;
```

## 初始化

```c
SqList crateSqList(void) {
    SqList l;
    l.length = 0;
    return l;
}
```

## 按元素值查找

```c
int findElem(SqList l, int e) {
    int i;
    for(i=0; i<l.length; i++) {
        if (e==l.data[i]) {return i;}
    }
    return -1; //返回-1作为查找失败标志
}
```

## 插入元素

```c
bool insert(SqList *l, int index, int data) {
    if((index < 1) || (index > (l->length + 1)) || (l->length==MAXSIZE)) {return false;}
    int i;
    for(i=l->length-1; i>=index; i--) {
        l->data[i+1]=l->data[i];
    }
    l->data[index-1]=data;
    l->length++;
    return true;
}
```

* 索引也可以从0开始
* 插入元素的操作最好情况是直接在末尾插入 $O(1)$, 最坏为表头插入 $O(n)$
* 平均情况:在长度为n的顺序中,任意插入一个位置i, $\frac{1}{n+1}*(1+2+3+...+n)=\frac{n}{2}$ 即 $O(n)$

## 删除元素

```c
bool delete(SqList *l, int index, int *output) {
    if ((index < 1) || (index > l->length)) {return false;}
    output=l->data[index-1];

    int i;
    for(i=index; i < l->length; i++) {
        l->data[i-1]=l->data[i];
    }
    l->length--
    return true;
}
```

## 按索引求值

```c
bool getItem(SqList l, int index, int *output) {
    if(index<1 || index>l.length) { return false; }
    *output = l.data[index-1];
    return true;
}
```
