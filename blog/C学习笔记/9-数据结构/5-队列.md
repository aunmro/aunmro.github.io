# 队列

先进先出

## 顺序队列

### 结构体

```c
#include <stdbool.h>
#define MAXSIZE 100

typedef int Element;

typedef struct SqQueue {
    Element data[MAXSIZE];
    int front, rear;
} SqQueue;
```

### 判断队空

```c
bool SqQueueIsEmpty(SqQueue q) {
    if (q.front == q.rear) {
        return true;
    }
    else {
        return false;
    }
}
```

### 入队

* **先加元素再移动指针**

```c
bool EnQueue(SqQueue* queue, Element value) {
    if (queue->rear == MAXSIZE) { return false; } /*队列已满*/
    queue->data[queue->rear] = value;
    queue->rear = queue->rear + 1;
    return true;
}
```

### 出队

```c
bool DeQueue(SqQueue* q, Element* e) {
    if (q->front == q->rear) { return false; } /*空队列*/
    *e = q->data[q->front];
    q->front = q->front + 1;
    return true;
}
```
