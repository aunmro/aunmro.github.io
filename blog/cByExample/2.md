# eg2

$摄氏度C=\frac{5}{9}(华氏度F-32)$,编写一个程序打印

| 摄氏度 | 	华氏温度 |
| ------ | -------- |
|        |           |

```c
#include <stdio.h>

/*打印摄氏度和华氏温度*/
/* 被包內的地方是注释 */

int main(void) {
    int fahr, celsius;
    int lower, upper, step;
    /*
      声明变量 int fahr
      1. 变量需要先声明后使用
      2. int=>变量类型：整数
      3. 其它类型
         1. float=>浮点数
         2. char=>一个字符
         3. short=>短整数
         4. double=>双精度
     */

    lower = 0;
    upper = 300;
    step  = 20;
    /*为变量赋值*/

    fahr = lower;

    /*
      while循环=>在条件内一直执行语句
      while(条件) {
           执行语句
      }
      被{}包含的需要缩进
    */
    while (fahr <= upper) {
        celsius = 5*(fahr-32)/9;
        printf("%d\t\%d\n", fahr, celsius);
        fahr = fahr + step;
    }

    return 0;
}
```
