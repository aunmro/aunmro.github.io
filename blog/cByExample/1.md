# 例子1

例子1: 编写程序输出"hello world"

C语言是编译型语言, 编写一个C程序的步骤:

1. 使用编辑器(vscode等)编写程序编写程序源代码
2. 使用编译器将源代码编译为计算机识别的机械码(010100之类)

## 代码

> 输出 "Hello World"

```c
# include <stdio.h>

main() {
    printf("hello world");
}
```

## 编译程序

运行`gcc eg1.c`, 执行后会产生 `a.out`, 这就是你写的程序编译后形成的. 运行程序 `./a.out`, 运行结果`

```
hello world
```

## 分析这个程序

```c
# include <stdio.h>

main() {
    printf("hello world\n");
}
```

1. C 程序都是由函数和变量组成. 变量用于存储计算过程中的值; 函数中包含语句, 以执行某些操作
2. `#include <stdio.h>` 引入标准库, IO库负责输入/输出的工作
3.  `main()` main 函数, 程序的入口,  一切程序从main开始运行. main 函数会调用其他函数来完成工作.
4.  main函数中的执行语句被包含在`{}`里
5. `printf("hello world\n");` 调用printf函数输出被引号包围的字符串"hello world"
    * 被双引号`""`包裹起来的字符被称为字符串, 比如"hello world\n"就是字符串
    * `\n`表示换行, 从下一行左侧开始输出

        `\n` 这种带`\`表示转义序列, 并不会被输出, 其他的转义序列还有:

        | 转移符号 |   含义    |
        | ------- | -------- |
        | \t      | 制表符    |
        | \b      | 退格符    |
        | \"      | 双引号    |
        | \\      | 表示\本身 |

    * `printf`函数不会自动换行, 例子1中的代码等同于以下代码

        ```c
         # include <stdio.h>

         main() {
             printf("hello ");
             printf("world");
             printf("\n");
         }
        ```

7. 调用函数时, 可以附加参数.
    *  执行`printf()`时, `"hello world\n"` 就是参数
    *  而`main()`函数则没有参数

# 例子2

$摄氏度C=\frac{5}{9}(华氏度F-32)$,编写一个程序打印

| 摄氏度 | 	华氏温度 |
| ------ | -------- |
| 0      | -17       |
| 20     | -6        |
| 40     | 4         |
| 60     | 15        |
| 80     | 26        |
| 100    | 37        |
| 120    | 48        |
| 140    | 60        |
| 160    | 71        |
| 180    | 82        |
| 200    | 93        |
| 220    | 104       |
| 240    | 115       |
| 260    | 126       |
| 280    | 137       |
| 300    | 148       |

```c
#include <stdio.h>

/* 当fahr=0,20,...,300时,打印摄氏度和华氏温度对照表*/

main() {
    int fahr, celsius;
    int lower, upper, step;

    lower = 0;
    upper = 300;
    step  = 20;
    /*为变量赋值*/

    fahr = lower;

    /*
      while循环=>在条件内一直执行语句
      while(条件) {
           执行语句
      }
      被{}包含的需要缩进
    */
    while (fahr <= upper) {
        celsius = 5*(fahr-32)/9;
        printf("%d\t\%d\n", fahr, celsius);
        fahr = fahr + step;
    }

    return 0;
}
```

## 注释

`/* 当fahr=0,20,...,300时,打印摄氏度和华氏温度对照表*/` 被 `/**/` 包起来的部分是注释，会被编译器忽略。
