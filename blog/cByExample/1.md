# 例子1

例子1: 编写程序输出"hello world"

C语言是编译型语言, 编写一个C程序的步骤:

1. 使用编辑器(vscode等)编写程序编写程序源代码
2. 使用编译器将源代码编译为计算机识别的机械码(010100之类)

## 例子1代码

```c
# include <stdio.h>

int main(void) {
    printf("hello world");
    return 0;
}
```

1. `#include <stdio.h>` 引入标准库
2.  main 函数
    `int main() { ... }` 	main函数是程序的入口, 一切程序从main开始运行
3. `printf()`打印函数, 在命令行中输出 hello world
4.   "" 双引号包含中的被称为字符串, `"hello world\n"`
5. `\n`表示换行

## 编译程序

1. 命令: gcc eg1.c, 执行后产生 a.out. 运行程序 ./a.out
2. 为你的程序命名, gcc eg1.c -o eg1

    -o 程序名
