# eg 1

```c
# include <stdio.h>

int main(void) {
    printf("hello world");
    return 0;
}
```

1. `#include <stdio.h>` 引入标准库
2.  main 函数
    `int main() { ... }` 	main函数是程序的入口, 一切程序从main开始运行
3. `printf()`打印函数, 在命令行中输出 hello world
4.   "" 双引号包含中的被称为字符串, `"hello world\n"`
5. `\n`表示换行

## 编译程序

> *C语言是编译型语言, 编译器将C语言源代码编译为机械码(机器理解的0,1码)*

1. 命令: gcc eg1.c, 执行后产生 a.out. 运行程序 ./a.out
2. 为你的程序命名, gcc eg1.c -o eg1

    -o 程序名
