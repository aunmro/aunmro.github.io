函数定义

函数对微积分来说就像是“建筑材料”一样，那么什么是函数？

什么是函数

\(y=f(x)\)

函数: 将一个对象x转化为另一个对象y的规则, 上式子中 f 就是一个函数。

  1. 起始对象x称为输入, 来自称为 定义域 的集合
  2. 返回对象y称为输出, 来自称为 值域 的集合
  3. 一个函数必须给 每一个有效的输入指定唯一的输出

怎么区分2个函数

举一个例子：

例子 \(f(x)=x^2\) 和 \(g(x)=x^2(x \ge 0)\)

  • \(f(x)\) 定义域，值域范围都为 \(R\)
  • \(g(x)\) 的定义域范围为非负数, 值域范围是非负数的集合

g 和 f 有相同的规则, 但 g 的定义域小于 f 的定义域, g 是 限制 f 的定义域产生的

确定函数2个要素: 定义域和对应关系(对应关系决定值域),确定函数相同需要二者都相同

那如何表示定义域，值域的范围呢 => 区间表示法

区间表示法

区间用来表示数值的范围, 比如 \(\left\{ x:2\ < x \leq 5 \right\}\), 表示为\((2, 5]\)

  • \([a, b]\) 为闭区间
  • \((a, b)\) 为开区间
  • \((a, b]\) 为半开区间

怎么求定义域

求定义域分为2种情况

  1. 函数的定义中包括定义域,比如:\(g(x)=x^2(x\ge 0)\)
  2. 定义域没给出时, 定义域包括实数集尽可能多的部分

    比如\(k=\sqrt x\) 定义域为 \([0, \infty )\)

常见几种定义域情况：

  1. 分母不为零.
  2. 不能取负数的平方根 (或四次根, 六次根, 等等).
  3. 不能取一个负数或零的对数

比如: \(tan(90°)\) 无意义 \(\frac{sin(90°)}{cos(90°)}=\frac{1}{0}\)

利用图像求值域

函数 \(F(x)=x^2\),定义域为 \([-2, 1]\), 利用图像求值域

画出函数图像, 从左右边朝 y 轴水平地射入两束亮光. 曲线会在 y 轴上有两个影子, 分别在 y 轴的左右侧. 值域就是影子的并集,即\([0, 4]\)

怎么判断一个图像是否是函数呢 => 垂线检验

垂线检验

函数 \(f\) 是所有坐标为 \((x, f (x))\) 的点的集合

以某个实数 x 开始

  1. 若 \(x\) 在定义域中,就画点 \((x, f (x))\), 这个点在 \(x\) 轴上的点 \(x\) 的正上下方, 高度为 \(f(x)\)
  2. \(x\) 不在定义域中, 不画任何点
  3. 对于每一个实数 \(x\), 重复这个过程构造出函数的图像

这里的关键是:

不可能有两个点有相同的 x 坐标 => 在图像上没有两个点会落在相对于 x 轴的同一条垂线上 => 垂线检验

有某个图像是否是函数的图像, 看它是否任何的垂线和图像相交多于一次

下图就不是一个函数图像

\(x^2+y^2=9\) 的函数图像则可以进行垂线检验

反函数

什么是反函数

反函数:对于 \(y=f(x)\) ,由 \(y\) 反推 \(x\) , 即 $x=f^{-1}\((y)\)

对于函数 \(f(x)=y\) , 其反函数 \(f^{-1}\)

  1. \(f^{-1}\) 的定义域和 f 的值域相同.
  2. \(f^{-1}\) 的值域和 f 的定义域相同.
  3. \(f^{-1}(y)\) 的值就是 \(f(x)=y\) 中的x

如果 \(f(x)=y\), 那么反函数 \(f^{-1}(y) = x\) (我们习惯写为 \(f^{-1}(x) = y\) )

求反函数时, 必须确定只有一个 \(x\) 值满足 \(y=f(x)\) ，即 \((0, y)\) 的水平线和函数图像仅有一次相交, 交点为点\((x, y)\) 。 其实还是 对于反函数输入值 \(y\) 的输出值 \(x\) 唯一

怎样检验函数的反函数是否存在 => 水平线检测

水平线检测

  1. 如果每一条水平线和一个函数的图像相交至多一次 => 函数有一个反函数
  2. 如果即使只有一条水平线和图像相交多于一次 => 函数就没有反函数

下图以 \(f(x)=x^3\) 和 \(g(x)=x^2\) 为例, 函数f有反函数, 函数g没有反函数

其他情况：

水平线根本就没有和曲线相交 => y 不在在值域当中
