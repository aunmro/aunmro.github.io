线性函数

\(f(x) = mx + b\) 它的图像是直线, 直线的斜率是 m

  1. m正数,线由左下向右上, m越大,斜率越大, "上坡"越陡
  2. m为负数,图像由左上向右下, 绝对值越大, "下坡"越陡
  3. m为0, 图像是一条水平直线

如何求线性函数的截距

  1. y 轴的截距 => 设 x = 0
  2. x 轴的截距 => y=0

点斜式是怎样的

  1. 直线通过 \((x_0, y_0)\) , 斜率为m, 它的公式为 \(y=y_0=m(x-x_0)\)
  2. 直线通过 \((x_1,y_1),(x_2,y_2)\) , 它的斜率为 \(\frac{y_2-y_1}{x_2-x_1}\)

多项式

基于 x 的非负次幂乘以实数, 然后把有限个这样的项加到一起, 比如 \(f(x)=5x^4-4x^3+10\)

  1. 数学通项公式: \(f(x)=a_nx^n+a_{n-1}x^{n-1}+...+a_2x^2+ax+a_0\)
  2. 多项式次数: 最大项的次数 \(f(x)=5x^4-4x^3+10\) 的次数是4
  3. 单个多项式的图像

二次函数

\(p(x) = ax^2 + bx + c\)

判别式

根据判别式的符号可以判断二次函数到底有两个、一个还是没有实数解

\(\Delta = b^2-ac\)

  1. \(\Delta>0\) 2个不同解
  2. \(\Delta=0\) 2个相同解
  3. \(\Delta<0\) 无解

解为 \(x = \frac{-b\pm \sqrt{b^2-4ac}}{2a}\)

指数函数

\(y=a^x (a>1)\) .图像如下图(以 \(y=2^x\) 为例)

  1. 定义域为全体实数
  2. y轴截距为1, 因为 \(a^0=1\)

对数函数

指数函数的反函数是对数函数

\(f(x)=log_a{x}\)

  1. 定义域 \([0, +\infty)\)
  2. 值域 全体实数
  3. y轴为渐进轴

绝对值函数

\(f(x)=|x|\)

\(|x|=\begin{cases} x(x\ge 0)\\ -x(x<0) \end{cases}\)

绝对值函数表示数轴上 0 和 x 的距离，绝对值函数的图像是以x轴一下的部分做关于x轴的镜像

三角函数

正弦 \(sin(\theta)=\frac{对边}{斜边}\)

余弦 \(cos(\theta)=\frac{邻边}{斜边}\)

正切 \(tan(\theta)=\frac{对边}{邻边}\)

倒函数

余割 \(csc(x)=\frac{1}{sin(x)}\)

正割 \(sec(x)=\frac{1}{cos(x)}\)

余切 \(cot(x)=\frac{1}{tan(x)}\)
