怎样弄一个Java开发环境

开发环境

  • jdk
    1. 下载jdk
    2. 设置 JAVA_HOME 环境变量
    3. jdk/bin 放入环境变量 path
  • eclipse
    1. 下载
    2. 使用开源镜像汉化 Exlipse汉化

Hello World

  1. Eclipse中创建项目 => 项目名 firstblood
  2. 创建类：文件=>新建=>类
    1. 包：类所在的包，包名一般是公司域名的倒置，可以没有。这里选择=com.firstblood=
    2. 名称：类名 HelloWorld
    3. 创建方法存根 public staic void main(String[] args)
  3. 自动生成了代码，在最中央加入 System.out.print("Hello World."); 
    package com.firstblood;

 public class HelloWorld {
     public static void main(String[] args) {
	 System.out.print("hello world");
     }
 }
  4. 点击运行

如何用命令行生成带目录的class文件

javac -d target.java

解释一下Hello World 

package com.firstblood;  //包名，即生成的目录

public class HelloWorld {   //java程序必须被包裹在类中
    public static void main(String[] args) {    //静态main方法，程序的入口
	System.out.print("hello world");  //java的输出流
    }
}

怎样声明变量 

package com.firstblood;

public class HelloWorld {
    int y; //成员变量

    public static void main(String[] args) {
	int x;
	float f = 4.5f;
	System.out.print(x); //错误,未初始化
	System.out.print(f);
    }
}

那常量呢 

package com.firstblood;

public class HelloWorld {
    public static final double PI = 3.14;  //HelloWorld.PI 直接访问

    public static void main(String[] args) {
	final int num = 10;
	System.out.println(PI);
	System.out.println(num);
    }
}
