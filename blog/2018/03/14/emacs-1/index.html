Emacs 自带教程脱水版

Emacs 常用的快捷键包括

  1. Ctrl，用 C 表示，比如 C-a
  2. Meta, 即 Alt，用 M 表示，比如 M-x
  3. Enter, 用 RET 表示

退出 Emacs，使用 C-x C-c

基本的光标操作

  1. C-v 和 M-v 向前、向后跳跃一个屏幕
  2. C-l 将光标所在行在屏幕中央显示

上下左右作为方向键，可能效率并不高，因为你的手指放在标准键位的。Emacs 中可以使用以下快捷键代替：

  1. C-p => 上，即 privious
  2. C-n => 下，即 next
  3. C-f => 左，即 forward
  4. C-b => 右，即 backward

比较一下 M 和 C

Emacs 中还可以一个字符一个字符的移动， M-b 和 M-f

  f b n p e a d
C 前一个字符 后一个字符 下一行 上一行 行尾 行首 删除一个字符
M 前一个单词 后一个单词     段尾或者函数尾 段首、函数首 删除一个单词

Emacs 还可以移动到文件的首尾 M-< M->

重复操作

Emacs 可以接受数字参数来达到重复操作的目的

C-u 5 C-p 向上移动5行，同样也可以用 M 8 c-p ,对于字符和单词操作都是如此
