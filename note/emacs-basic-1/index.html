1 基本操作

  1. 直接使用 Menu

1.1 打开，关闭，保存

key command 作用
Ctrl+x Ctrl+f find-file 打开文件
Ctrl+x Ctrl+s save-buffer 保存文件
Ctrl+x k kill-buffer 关闭当前buffer

1.2 选择文本

C-S-space 选择
C-x h 全选

1.3 复制，剪切，paste，撤销

key command 作用
Ctrl+/ undo 撤销
Alt+w kill-ring-save 复制
C-w kill-region 剪切
C-y yank 粘贴

1.4 移动光标

上下左右 移动
C + <- 前一个词
C + -> 后一个词

1.5 删除

Alt+D 删除一个单词
Alt+Backspace 向前删除一个词
C+k 删除一行

1.6 cua-mode

我们熟悉的 C-v C-c 这种

1.7 分割窗口

Ctrl+x 2 split-window-below
Ctrl+x 3 split-window-right
Ctrl+x 1 delete-other-windows
Ctrl+x o other-window

1.8 搜索

C-s C-r

1.9 命令

Emacs 中，所有按键都是命令。我们同样可以自己执行命令

Alt-x execute-extended-command
C-g keyboard-quit
C-h k describe-key
C-h f describe-function

1.10 buffer

ibuffer 进入 buffer 模式，buffer 模式有 operat 菜单，或者在 ibuffer 中使用键盘操作：

m Mark
u umark
*u Mark unsaved
S save marked buffer
D close marked buffer
